What is DevOps?
DevOps is a process of improving the app delivery by ensuring there is a proper automation with a code quality that is maintained and ensuring there is a continuous monitoring and testing in place

What is the difference between soft and hard link?
hard link creates a copy of a file, it will still exist if the original find was deleted
soft link will not be opened if the original file was deleted

What is the difference between break and continue statements
break - breaking the execution
continue - continuing the execution (skip this and continue the next)
